# **App Name**: Chain of Kindness

## Core Features:

- Wallet Connection: Connect to a wallet using Wagmi and RainbowKit for blockchain interaction.
- Give Help Form: A form that lets a user enter the recipient's address and a message, then record this information on the blockchain.
- Help Board: Display a list of all 'Help' records stored on the blockchain in an easily readable format.
- AI Reputation Leaderboard: An AI-powered reputation system to calculate and display a leaderboard of top helpers. The tool uses past Proof of Help data.

## Style Guidelines:

- Primary color: A warm and inviting blue (#5DADE2) to convey trust and support, drawing inspiration from the concept of community.
- Background color: A very light, desaturated blue (#F0F8FF), creating a calming and clean backdrop.
- Accent color: A gentle green (#A3E4D7), used for success messages and highlights, complementing the primary color.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable style, suitable for both headings and body text.
- Use simple, clear icons to represent different actions, creating intuitive and user-friendly interaction.
- The layout should be clean and organized, with a max-width container to focus user attention and enhance readability.
